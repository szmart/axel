The "rich content" plugin

The "rich content" plugin is a text editing field with stylable content. As it makes its handle contentEditable, it offers a perfect integration with the page to create a WYSIWYG experience. You can use it to edit text that flows around floating elements.

Parameters

  • noedit (true|false) : set it to true to make the field read only (constant).
  • class (name) : set a class name on the handle.
  • lang (name) : set the input/output format (default is `fragment').
  • multilines (normal|none) : allows for line breaks in the editable content.

Default configuration

The default configuration accepts an input composed of a list of Fragments and Links. Fragments contain a single text node, Links are parents to a LinkRef and a LinkText. The Fragments and LinkTexts can optionally bear a RichStyle attribute.

With spans

The `span' configuration accepts an input composed of a list of HTML spans and a nodes. spans and as contain a single text node, and can possess a RichStyle attribute.

With multilines

The `multilines' configuration will display the line breaks appearing in the content.
